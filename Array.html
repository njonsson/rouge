areAll

Boolean areAll(<Function> block)
    Passes each element in the array to block, returning true if block never returns false or null. If block is not given, Rouge adds an implicit block that will return true only if none of the elements is false or null. 
     var array = ['foo', 'bar', 'bizzle'];
 var result = array.areAll(function(item) {
   return item.length >= 3;
 });
 result // => true
 array  // => ['foo', 'bar', 'bizzle']
 
 var array = ['foo', 'bar', 'bizzle'];
 var result = array.areAll(function(item) {
   return item.length >= 4;
 });
 result // => false
 array  // => ['foo', 'bar', 'bizzle']
    Without the optional block: 
     var array = [null, true, 99];
 var result = array.areAll();
 result // => false
 array  // => [null, true, 99]
    Parameters:
      block - (optional) The function to execute. Should have one parameter
    Returns:
      true if block does not return false or null for any element
    See:
      - #areAny

areAny

Boolean areAny(<Function> block)
    Passes each element in the array to block, returning true if block ever returns a value other than false or null. If block is not given, Rouge adds an implicit block that will return true if any of the elements is not false or null.

 var result = array.areAny(function(item) {
   return item.length >= 3;
 });
 result // => true
 array  // => ['foo', 'bar', 'bizzle']

 var result = array.areAny(function(item) {
   return item.length <= 2;

 result // => false
 array  // => ['foo', 'bar', 'bizzle']
    Without the optional block: 
     var array = [null, true, 99];

 result // => true

    Parameters:

collect

Array collect(<Function> block)
    Creates a different array of the same size. Invokes block once for each element in the array, passing that element as an argument.

 var result = array.collect(function(item) {
   return item + '!';
 });
 result // => ['foo!', 'bar!', 'baz!']
 array  // => ['foo', 'bar', 'baz']
    Parameters:
      block - The function to execute. Should have one parameter
    Returns:
      An array containing the values returned by block

detect

Object detect(<Function> ifNone, <Function> block)
    Finds an element in the array. Invokes block once for each element in the array, passing that element as an argument. 
     var array = ['foo', 'bar', 'baz'];
 var result = array.collect(function(item) {
   return item === 'bar';
 });
 result // => 'bar'
 array  // => ['foo', 'bar', 'baz']
    With the optional ifNone argument: 
     var array = ['foo', 'bar', 'baz'];
 var ifNone = function() { return 'nothing here'; };
 var result = array.detect(ifNone, function(item) {
   return item === 'bizzle';
 });
 result // => 'nothing here'
 array  // => ['foo', 'bar', 'baz']
    Parameters:
      ifNone - (optional) A function that returns the value to be returned by #detect if there are no matches
      block - The function to execute. Should have one parameter and return either true or false
    Returns:
      The first element for which block returns true. If block never returns true and ifNone is specified, then the return value of a call to ifNone is used. If block never returns true and ifNone is not specified, then null is used
    See:
      - #find

inject

Object inject(<Object> initial, <Function> block)
    Combines the elements of the array by applying block to an accumulator value (memo) and each element in turn. Invokes block once for each element in the array, passing memo and that element as arguments. At each invocation of block, memo is set to the value returned by block previously. The first form lets you supply an initial value for memo. 
     var array = [1, 2, 3, 4];
 var result = array.inject(10, function(memo, item) {
   return memo * item;
 });
 result // => 240
 array  // => [1, 2, 3, 4]
    The second form uses the first element as the initial value (and skips that element while iterating). 
     var array = [1, 2, 3, 4];
 var result = array.inject(function(memo, item) {
   return memo * item;
 });
 result // => 24
 array  // => [1, 2, 3, 4]
    Parameters:
      initial - (optional) The initial value of the memo argument passed to block
      block - The function to execute. Should have two parameters (memo and the element) and return the new value of memo
    Returns:
      The value returned from the last invocation of block
