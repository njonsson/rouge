Rouge is a JavaScript implementation of some of the Ruby base classes and
modules.

h1. Usage

Rouge makes your JavaScript look and feel more like Ruby while preserving sound
JavaScript style.

Stop writing:

<pre><code>var array = ['a', 'b', 'c', 'd', 'e'];
for (var i = 0; i < array.length; i++) {
  alert(array[i]);
}</code></pre>

when instead you can write:

<pre><code>var array = ['a', 'b', 'c', 'd', 'e'];
array.each(function(s) {
  alert(s);
});</code></pre>
